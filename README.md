# 🚀 My Auth Service

A centralized authentication and authorization service built with **React** (frontend) and **Spring Boot** (backend).  
It provides secure user registration, login, JWT-based authentication, role management, and single sign-on (SSO) capabilities for multiple projects and applications.

---

## 🔑 Features
- Secure **user registration** and **login**
- **JWT-based authentication**
- **Role management** (admin, user, etc.)
- **Single sign-on (SSO)** support for multiple projects and applications

---

## 🌐 Live Demo
- **Backend API**: [my-auth-service-wf2a.onrender.com](https://my-auth-service-wf2a.onrender.com/)  
- **Frontend App**: [my-auth-service.vercel.app](https://my-auth-service-olsywxwho-oms-projects-76777c3b.vercel.app/)

---

## 🛠️ Tech Stack
- **Frontend**: React, Vercel (deployment)
- **Backend**: Spring Boot, Render (deployment)
- **Authentication**: JWT, Role-based access control

---

## 📖 Usage
1. Register a new account via the frontend.
2. Login to receive a JWT token.
3. Use the token to access protected routes in your applications.
4. Manage roles and permissions through the backend service.
